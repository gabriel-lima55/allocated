===== SISTEMA EM MANUTENÇÃO =====

Projeto Integrador – Sistema de Reserva de Salas de Estudo (IFRN) 

Prática profissional – Resolver o problema da dificuldade de reservar salas de estudos. Desenvolvemos um sistema que digitaliza o processo de solicitação e gestão de reservas entre alunos e gestores 

Entregamos um MVP (Minimum Viable Product) baseado em regras de negócio. Para isto realizamos um Troubleshooter com Desk Research e história de usuário, que nos orientou a criação de um sistema backend com API REST, banco de dados em nuvem e um protótipo navegável mobile em Figma. 

Stack: Figma, API REST, Banco de Dados em Nuvem.
